geocoder.geocoding.query

Interface to query for data.

This module defines the query methods for each table in the database.

geocoder.geocoding.query.data

The database is formed by numpy arrays. Each of them is identified by a name and accessible by this dictionary with data[name].

Type

dict of numpy.ndarray

geocoder.geocoding.query.limits

limits[table] stores the limits of the numpy array called table.

Type

dict of tuple of int

geocoder.geocoding.query.complete_voie_selection(code_postal, commune, voie)[source]

Select record on voie table with field normalise most similar to voie.

Search for voie in the entire voie table. This method is the option to find the voie once the method select_voie has failed. It first tries to find the voie belonging to the commune the most similar to commune and if this step fails it will try to find the voie belonging to a postal code similar to code_postal.

Parameters

  • code_postal (str) – The postcode.

  • commune (str) – The city name.

  • voie (str) – The street name.

Returns

The index of the record if the search was succeeded,

None otherwise.

Return type
